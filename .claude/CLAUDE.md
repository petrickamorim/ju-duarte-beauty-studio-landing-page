# Julia Duarte Beauty Studio — Contexto do Projeto

## Visão Geral

Este projeto é o site oficial da Julia Duarte Beauty Studio.

O projeto foi inicialmente desenvolvido em React + Vite, porém o objetivo é migrar completamente toda a aplicação para Angular.

O site deve transmitir:

- Elegância
- Sofisticação
- Feminilidade
- Aparência premium
- Modernidade
- Rapidez
- Excelente experiência mobile

O foco principal é conversão via WhatsApp.

---

# Stack Principal

## Frontend

- Angular 20+
- TypeScript
- SCSS
- Angular Signals
- Standalone Components
- Angular Router

## UI/UX

- Mobile-first
- Responsivo
- Alta qualidade visual
- SEO otimizado
- Acessibilidade

## Deploy

- Vercel
- Netlify

# Regras Gerais

## Arquitetura

- Código limpo
- Componentes reutilizáveis
- Estrutura escalável
- Evitar duplicação
- HTML semântico
- Componentes pequenos

## Angular

Sempre utilizar:

- Standalone Components
- TypeScript strict
- Signals quando fizer sentido
- Lazy loading quando necessário

Evitar:

- NgModules desnecessários
- Componentes gigantes
- CSS inline
- Código acoplado
